Context Manager

  • Context managers allow you to allocate and release resources precisely when you want to
  • The most widely used example of context managers is the 
    with
    statement
  • Below codes are technically equal
    with open('some_file', 'w') as f:
    f.write('Hola!')
    # File is automatically closed at this point



    f = open('some_file', 'w')  # Opens resource
f.write('Hola!')            # Carry out processing
f.close()                   # Closes the resource
